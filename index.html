  7. 7. What are stream class ?

    8. Stream classes in C++ facilitate input and output operations on files and other I/O devices. These classes have specific features to handle program input and output, making it easier to write portable code that can be used across multiple platforms.
    To use streams in C++, you need to include the appropriate header file. For instance, to use input/output streams, you would include the iostream header file. This library provides the necessary functions and classes to work with streams, enabling you to read and write data to and from files and other I/O devices.
    The main stream classes in C++ are:

    1. istream (input stream):
    This class is used for input operations. It allows reading data from various sources, such as files, keyboard input, etc.

    2. ostream (output stream):
    This class is used for output operations. It allows writing data to various destinations, such as files, console output, etc.

    3. iostream:
    This class is a combination of both "istream" and "ostream" classes, enabling both input and output operations on the same stream object.

    4. ifstream (input file stream):
    This class is derived from "istream" and specifically designed for reading data from files.

    5. ofstream (output file stream):
    This class is derived from "ostream" and specifically designed for writing data to files.

    6. stringstream:
    This class is derived from "iostream" and operates on strings in memory. It allows treating a string as a stream, enabling input and output operations.

    These stream classes work with stream objects, which are instances of the respective classes. Stream objects are used to interact with various data sources and perform input and output operations using the insertion (<<) and extraction (>>) operators.

  4. 4. What is dynamic memory allocation ?

    5. The process by which storage or memory space allocated to the variable during the runtime of a program is called dynamic memory allocation. Memory can be allocated in two ways: static and dynamic.

    1 . Static allocation or compile-time allocation - Static memory allocation means providing space for the variable. The size and data type of the variable is known, and it remains constant throughout the program.

    2 . Dynamic allocation or run-time allocation - Dynamic memory allocation occurs during the program's execution, allowing you to allocate memory at runtime when the size or number of elements is not known in advance. This type of memory allocation is usually performed on the heap. .

    Dynamic memory allocation is achieved using functions like malloc, calloc, realloc (C), or new (C++).

  5. 5. What is spare matrix ?

    6. Sparse matrices are those matrices that have the majority of their elements equal to zero. In other words, the sparse matrix can be defined as the matrix that has a greater number of zero elements than the non-zero elements.

    There are the following benefits of using the sparse matrix -

    Storage - We know that a sparse matrix contains lesser non-zero elements than zero, so less memory can be used to store elements. It evaluates only the non-zero elements.

    Computing time: In the case of searching in sparse matrix, we need to traverse only the non-zero elements rather than traversing all the sparse matrix elements. It saves computing time by logically designing a data structure traversing non-zero elements
