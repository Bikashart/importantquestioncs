  2. 2. What are object in c++ ?

    3. An object is an instance of a class. It is a real variable or entity that exists in memory and can be used to access the member variables and member functions defined in the class.When a class is defined no memory is allocated, but when an object is created memory is allocated. To use data and its member function we can declare objects.

    Syntax for declaring objects :
    ClassName objectName;
